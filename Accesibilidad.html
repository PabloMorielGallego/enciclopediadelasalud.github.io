Accesibilidad

1. Identificación del idioma principal de la página

El idioma principal de esta página es el español, indicado mediante el atributo lang="es" en la etiqueta <html>. Esto permite a los usuarios de tecnologías de asistencia identificar el idioma y ajustar la pronunciación adecuadamente.

2. Etiquetado semántico de la página

Se ha utilizado una estructura semántica adecuada, empleando etiquetas como <header>, <nav>, <main>, <section> y <footer> para organizar el contenido de forma lógica y accesible. Esto facilita la navegación de los usuarios, especialmente de aquellos que utilizan lectores de pantalla.

3. Texto alternativo de las imágenes

Todas las imágenes de esta página incluyen un texto alternativo relevante mediante el atributo alt, el cual describe de manera adecuada el contenido visual. Esto permite que los usuarios con discapacidades visuales puedan comprender la información contenida en las imágenes.

4. Mecanismos de navegación implementados

Se ha implementado un menú de navegación accesible utilizando la etiqueta <nav> para facilitar el acceso a las diferentes secciones del sitio. Además, se han incluido atajos de teclado para una navegación más rápida y eficiente.

5. Características adicionales de accesibilidad

  • La página es completamente accesible a través del teclado, lo que permite que los usuarios naveguen sin necesidad de un ratón.
  • El contenido tiene un contraste adecuado entre el texto y el fondo, lo que mejora la legibilidad y facilita la lectura para personas con discapacidades visuales, como la dislexia o baja visión.
  • Los formularios incluyen etiquetas <label> asociadas a los campos de entrada para facilitar su uso por lectores de pantalla y mejorar la experiencia de los usuarios con dificultades motoras o visuales.
  • Se ha implementado un diseño responsivo que permite que el contenido se ajuste correctamente en dispositivos móviles y pantallas de diferentes tamaños.
  • La página incluye controles de ajuste de tamaño de texto y la opción de cambiar el esquema de colores para mejorar la legibilidad según las necesidades individuales del usuario.
  • El contenido multimedia, como videos, incluye subtítulos y transcripciones cuando es necesario, para asegurar que los usuarios con discapacidades auditivas tengan acceso a la información.

6. Evaluación de la accesibilidad

Hemos realizado pruebas internas y externas utilizando herramientas de validación de accesibilidad, como el validador WAVE, para asegurar que la página cumpla con los criterios de accesibilidad establecidos por las normas internacionales, incluidas las WCAG 2.1. Sin embargo, estamos comprometidos con la mejora continua y estamos trabajando para resolver cualquier posible limitación identificada en futuras actualizaciones.

7. Observaciones y datos de contacto
