Accesibilidad

1. Identificación del idioma principal de la página

El idioma principal de esta página es el español, indicado mediante el atributo lang="es" en la etiqueta <html>.

2. Etiquetado semántico de la página

Se ha utilizado una estructura semántica adecuada, empleando etiquetas como <header>, <nav>, <main>, <section> y <footer> para organizar el contenido de forma lógica y accesible.

3. Texto alternativo de las imágenes

Todas las imágenes de esta página incluyen un texto alternativo relevante mediante el atributo alt.

4. Mecanismos de navegación implementados

Se ha implementado un menú de navegación accesible utilizando la etiqueta <nav> para facilitar el acceso a las diferentes secciones del sitio.

5. Características adicionales de accesibilidad

  • La página es totalmente accesible a través del teclado.
  • El contenido tiene un contraste adecuado para facilitar la lectura.
  • Los formularios incluyen etiquetas asociadas a los campos de entrada para facilitar su uso por lectores de pantalla.
